示范1

`http://127.0.0.1:7899/?cmd=get&t=日k:000021:20260423`

即可查看000021，20260423日的数据 json

示范2

`http://127.0.0.1:7899/?cmd=get&t=日k&k1=key:000021&k2=fwd:20260410,20260423`

即可查看000021，20260410**到**20260423日的kv数据

`http://127.0.0.1:7899/?cmd=vals&t=日k&k1=key:000021&k2=fwd:20260410,20260423`
